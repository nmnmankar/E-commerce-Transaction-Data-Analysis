# 📊 E-commerce Transaction Data Analysis

## 📌 Project Overview

This project analyzes an **E-commerce transaction dataset** to uncover insights about sales performance, customer purchasing behavior, and product category revenue.

The analysis was performed using **Python (Pandas, Matplotlib, Seaborn) in Jupyter Notebook**, and the results were visualized in an **interactive Power BI dashboard**.

The goal of this project is to transform raw transaction data into **actionable business insights** that can help improve sales strategy and decision-making.

# 🔍 Data Analysis Process

## 1️⃣ Data Cleaning

* Removed missing or inconsistent values
* Converted timestamps to proper date format
* Checked for duplicates and invalid entries
* Prepared data for analysis

## 2️⃣ Exploratory Data Analysis

Key questions explored:

* How are transaction amounts distributed?
* What are the sales trends over time?
* Which product categories generate the most revenue?
* Which countries contribute most to revenue?

---

# 📊 Power BI Dashboard

The Power BI dashboard provides an **interactive visualization of the analysis**.

### Dashboard Features

* Transaction amount distribution
* Revenue by product category
* Sales trend over time
* Customer purchase behavior by country
* Filters for month, country, and category

These visualizations help quickly identify **sales patterns and business opportunities**.

---

# 📈 Key Insights

* Certain product categories generate significantly higher revenue than others.
* Sales show noticeable fluctuations across months.
* A few countries contribute the majority of transactions.
* Lower transaction ranges contain the highest number of purchases.

These insights can help businesses **optimize marketing and product strategies**.

---

# 💡 Strategy Recommendations

* Focus marketing campaigns on **high-performing product categories**
* Target **top purchasing countries** for promotions
* Improve engagement during **low sales months**
* Use pricing strategies to increase higher-value transactions

# ▶️ How to Run the Project

### Run the Notebook

1. Download the repository
2. Open **Notebook.ipynb** in Jupyter Notebook
3. Install required libraries:

```
pip install pandas matplotlib seaborn
```

4. Run all cells to reproduce the analysis.

---

### Open the Dashboard

1. Install **Power BI Desktop**
2. Open the file:

```
Dashboard.pbix
```

3. Interact with filters and charts.
